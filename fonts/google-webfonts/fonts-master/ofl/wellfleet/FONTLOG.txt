This file provides detailed information on the Wellfleet font Software.
This information should be distributed along with the Wellfleet font
and any derivative works.

Basic Font Information

Wellfleet is a versatile low contrast slab serif text typeface 
with a a bouncy upbeat feeling. Wellfleet was inspired by German 
poster lettering. Despite having display letters as a source of 
inspiration Wellfleet is functional in a wide range of sizes.

Wellfleet is a Unicode typeface family that supports languages that
use the Latin script and its variants, and could be expanded to support 
other scripts.

More specifically, this release supports the following Unicode ranges: 
Latin-1, Latin-2: Easern Europe, Turkish, Macintosh Character Set

To contribute to the project contact Eben Sorkin at sorkineben@gmail.com
