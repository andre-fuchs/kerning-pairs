FONTLOG for the Alex Brush font

This file provides information on the Alex Brush Font Software.
This information should be distributed along with the Alex Brush 
fonts and any derivative works.

Basic Font Information

Alex Brush is a beautifully flowing brush script. It has short 
ascenders and descenders allowing a legibility not seen in 
other script fonts.

Alex Brush is a Unicode typeface that supports languages that
use the Latin script and its variants, and could be expanded 
to support other scripts.

This release supports the following Unicode ranges: 
Latin-1.
