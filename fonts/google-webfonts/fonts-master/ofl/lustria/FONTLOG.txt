FONTLOG for the Lustria fonts

This file provides detailed information on the Lustria Font Software.
This information should be distributed along with the Lustria fonts
and any derivative works.

Basic Font Information

Lustria is a rounded-serif text typeface with details that make it 
interesting to use at larger display sizes as well. Started in 1999, 
this typeface has been aged to perfection and is finally ready 
for public consumption.

Documentation can be found at http://www.madtype.com

To contribute to the project contact Matthew Desmond 
at mattdesmond@gmail.com
