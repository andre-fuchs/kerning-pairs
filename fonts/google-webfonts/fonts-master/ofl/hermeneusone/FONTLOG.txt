Basic Font Information

Hermeneus is a Grecian style font. It condensation lets you fit more 
characters in a line of text, giving your paragraphs more efficiency 
at the moment of composing them. It is intended to be read at small 
text sizes, but also works on large headers and titles.

Join the project at: http://www.impallari.com/projects/overview/hermeneus

ChangeLog

10 Nov 2011  (Pablo Impallari, Rodrigo Fuenzalida, Igino Marini) Hermeneus v1.2
- Hinted using the latest version of TTFAutohint, currently v0.9.3
- Even smaller file size for faster loading on the web

10 Nov 2011  (Pablo Impallari, Rodrigo Fuenzalida, Igino Marini) Hermeneus v1.1
- Fixed Vertical Metrics, Font Info and other small bugs
- Removed the KERN table, now we are using GPOS based Kerning.
- Smaller file size for faster loading on the web

26 Jun 2011  (Pablo Impallari, Rodrigo Fuenzalida, Igino Marini) Hermeneus v1.0
- Version 1 Release
- iKerned
- Fine tuned glyphs and OT features added

08 Jun 2012 (Pablo Impallari, Rodrigo Fuenzalida) Hermeneus v0.007
- Hermeneus is ready for iKern.  

16 May 2012 (Pablo Impallari, Rodrigo Fuenzalida) Hermeneus v0.001
- The first draft of the lowercase characters is done.
