Basic Font Information

Chau Philomene One is one of four families within the Chau 
superfamily. He is hardy, ideal for headings and highlighted 
text. His narrow letters and sharp angles give him a 
distinctive personality.

Chau Philomene One is a Unicode typeface family that supports 
languages that use the Latin script and its variants, and could be 
expanded to support other scripts.

More specifically, this release supports the following 
Unicode ranges: Latin-1

There are 4 Source files:

1. ChauPhilomeneOne-Regular-OTF.vfb Merged contours and 
   optimized file corresponding to the OTF file

2. ChauPhilomeneOne-Regular-TTF.sfd TrueType outlines with 
   hinting adjustments corresponding to the TTF file

3. ChauPhilomeneOne-Bold-OTF.vfb Merged contours and 
   optimized file corresponding to the OTF file

4. ChauPhilomeneOne-Bold-TTF.sfd TrueType outlines with 
   hinting adjustments corresponding to the TTF file
