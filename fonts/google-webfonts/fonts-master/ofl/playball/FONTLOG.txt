FONTLOG for the Playball font

This file provides information on the Playball Font Software.
This information should be distributed along with the Playball fonts
and any derivative works.

Basic Font Information

Playball is connecting script, perfect for
festive occassions and sporty events. 

Playball is a Unicode typeface that supports languages that
use the Latin script and its variants, and could be expanded 
to support other scripts.

This release supports the following Unicode ranges: 
Latin-1.
