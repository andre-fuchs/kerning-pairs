FONTLOG for the Italianno font

This file provides information on the Italianno Font Software.
This information should be distributed along with the Italianno fonts
and any derivative works.

Basic Font Information

Italianno is an elegant calligraphic script with clean 
connectors and classic forms. Perfect for warm, inviting 
situations.

Italianno is a Unicode typeface that supports languages that
use the Latin script and its variants, and could be expanded 
to support other scripts.

This release supports the following Unicode ranges: 
Latin-1.
