FONTLOG for the Viga font

This file provides detailed information on the Viga 
font Software.

This information should be distributed along with the 
Viga fonts and any derivative works.

Basic Font Information

Viga is a sanserif with a good performance on screen. 
Its anatomy gives it a great personality and also 
makes it useful for reading on screen. 

Viga is a Unicode typeface family that supports 
languages that use the Latin script and its variants, 
and could be expanded to support other scripts.

More specifically, this release supports the following 
Unicode ranges: Latin-1

To contribute to the project contact Alejandro Lo Celso 
at info@fontstage.com
