FONTLOG for the Ruthie font

This file provides information on the Ruthie Font Software.
This information should be distributed along with the Ruthie fonts
and any derivative works.

Basic Font Information

Ruthie is highly stylized and elegant calligraphic script.
It has ornate caps and lovely flowing lower case forms.

Ruthie is a Unicode typeface that supports languages that
use the Latin script and its variants, and could be expanded 
to support other scripts.

This release supports the following Unicode ranges: 
Latin-1.
