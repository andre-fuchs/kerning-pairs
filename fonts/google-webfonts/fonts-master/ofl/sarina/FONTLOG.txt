FONTLOG for the Sarina font

This file provides detailed information on Sarina 
Font Software.This information should be distributed along 
with the Sarina font and any derivative works.

Basic Font Information

Sarina is a display typeface with brush style letterforms. 
Sarina's medium contrast and wide setting offers a casual 
breezy feeling. Sarina is appropriate for medium to 
larger sizes.

Sarina is a Unicode typeface family that supports languages 
that use the Latin script and its variants, and could 
be expanded to support other scripts.

More specifically, this release supports the following 
Unicode ranges: Latin-1, Latin-2: Easern Europe, Turkish, 
Macintosh Character Set
