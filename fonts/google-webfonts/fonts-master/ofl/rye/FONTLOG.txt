FONTLOG for the Rye font

This file provides detailed information on Rye Font Software.This information should be distributed along with the Rye font and any derivative works.

Basic Font Information

Rye's bold attention getting shapes are useful for advertising. Rye is a medium contrast design inspired by posters using wood type. It gives off a feeling of the American West. It is suitable for use in medium to large sizes including headlines. This font was made specifically to be web type.

Rye is a Unicode typeface family that supports languages that use the Latin script and its variants, and could be expanded to support other scripts.

More specifically, this release supports the following Unicode ranges: Latin-1

There are three Source files:

1. Rye-Regular.vfb Original Source Files with 
   contour overlaps

2. Rye-Regular-otf.sfd Merged contours and 
   optimized file corresponding to the OTF file

3. Rye-Regular-ttf.sfd TrueType outlines with 
   hinting adjustments corresponding to the TTF file
