Khula (खुला) is a contemporary text Devanagari typeface family designed by Erin McLaughlin as a compliment to Open Sans, a Latin family designed by Steve Matteson. Currently it has 5 weights and supports Hindi.

This project is led by Erin McLaughlin, a type designed based in Wichita, USA. To contribute, see Khula on GitHub.
