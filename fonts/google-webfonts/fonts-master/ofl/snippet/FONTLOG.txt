FONTLOG for Snippet fonts

This file provides detailed information on the Snippet Font Software.
This information should be distributed along with the Snippet fonts
and any derivative works.

Basic Font Information

Snippet’s light weight and unusual construction give texts a special
flow and make you want to look twice! In headlines Snippet’s many 
details show and help you create a clean but witty look.

Snippet is a Unicode typeface family that supports languages that
use the Latin script and its variants, and could be expanded to 
support other scripts.
