FONTLOG for the Miltonian fonts

This file provides detailed information on the Miltonian Font Software.
This information should be distributed along with the Miltonian fonts
and any derivative works.

Basic Font Information

Miltonian is a Unicode typeface family that supports languages that
use the Latin script and its variants, and could be expanded to support other
scripts.

More specifically, this release supports the following Unicode ranges: Latin-1

Documentation can be found at www.impallari.com/miltonian
To contribute to the project contact Pablo Impallari at impallari@gmail.com
