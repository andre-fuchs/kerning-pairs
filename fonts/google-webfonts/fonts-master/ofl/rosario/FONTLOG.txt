FONTLOG for the Rosario fonts

This file provides detailed information on the Rosario Font Software.
This information should be distributed along with the Rosario fonts
and any derivative works.

Basic Font Information

Rosario is the name of the city of its type designer, 
Hector Gatti. This is a new typeface of classic proportions, 
subtle contrast and soft endings. Carefully produced, 
elegant, ideal for magazines and academic journals.

Rosario is a Unicode typeface family that supports languages that
use the Latin script and its variants, and could be expanded to 
support other scripts.

More specifically, this release supports the following 
Unicode ranges: Latin-1
