FONTLOG for the Plaster font

This file provides detailed information on Plaster Font Software.This information should be distributed along with the Plaster font and any derivative works.

Basic Font Information

Plaster is a Unicode typeface family that supports languages that use the Latin script and its variants, and could be expanded to support other scripts.

More specifically, this release supports the following Unicode ranges: Latin-1, Latin-2: Easern Europe, Turkish, Macintosh Character Set

To contribute to the project contact Eben Sorkin at sorkineben@gmail.com
