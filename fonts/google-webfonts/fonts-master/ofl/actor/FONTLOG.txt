FONTLOG for the Actor font

This file provides detailed information on Actor Font Software.
This information should be distributed along with the Actor 
font and any derivative works.

Basic Font Information

Actor is a Unicode typeface family that supports languages that 
use the Latin script and its variants, and could be expanded 
to support other scripts.

More specifically, this release supports the following 
Unicode ranges: Latin-1
