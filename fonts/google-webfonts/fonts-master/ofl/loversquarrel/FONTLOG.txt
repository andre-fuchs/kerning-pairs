FONTLOG for the Lovers Quarrel font

This file provides information on the Lovers Quarrel Font Software.

This information should be distributed along with the 
Lovers Quarrel fonts and any derivative works.

Basic Font Information

Lovers Quarrel is an ornate calligraphic script. It has 
beautifully embellished uppercase characters and clean, 
legible lowercase forms.

Lovers Quarrel is a Unicode typeface that supports languages 
that use the Latin script and its variants, and could be expanded 
to support other scripts.

This release supports the following Unicode ranges: 
Latin-1.
 
There are 2 Source files:

1. LoversQuarrel-Regular-OTF.vfb Merged contours and 
   optimized file corresponding to the OTF file

2. LoversQuarrel-Regular-TTF.sfd TrueType outlines with 
   hinting adjustments corresponding to the TTF file
