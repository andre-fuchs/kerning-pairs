FONTLOG for the Supermercado font

This file provides detailed information on Supermercado Font Software.This information should be distributed along with the Supermercado font and any derivative works.

Basic Font Information

Supermercado is a Unicode typeface family that supports 
languages that use the Latin script and its variants, 
and could be expanded to support other scripts.

More specifically, this release supports the following 
Unicode ranges: Latin-1, Latin-2: Easern Europe, Turkish, 
Macintosh Character Set.

There are three Source files:

1. Supermercado-Regular.vfb Original Source Files with 
   contour overlaps

2. Supermercado-Regular-OTF.vfb Merged contours and 
   optimized file corresponding to the OTF file

3. Supermercado-Regular-TTF.sfd TrueType outlines with 
   hinting adjustments corresponding to the TTF file

ChangeLog

15 April 2011 (Eben Sorkin/James Grieshaber) Supermercado v1.002
- Added support for full Latin-1, Latin-2: Easern Europe, 
Turkish, Macintosh Character Set.
- Mastered Font from Fontlab VBF to TTF and OTF
- Added true type hinting using TTF Autohint version 0.8

20 October 2011 (Eben Sorkin) Supermercado v1.001
- Adjusted spacing for web browsers.
- Mastered VFB to TTF.

14 October 2011 (James Grieshaber) Supermercado v1.000
- Completed first complete version of Supermercado in Fontlab ( VBF format )
