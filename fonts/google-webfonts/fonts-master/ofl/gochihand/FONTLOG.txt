FONTLOG for Gochi Hand font

This file provides detailed information on the Gochi Hand 
Font Software.

This information should be distributed along with the 
Gochi Hand fonts and any derivative works.

Basic Font Information

Gochi Hand is a Unicode typeface family that supports languages 
that use the Latin script and its variants, and could be 
expanded to support other scripts.

More specifically, this release supports the following 
Unicode ranges: Latin-1

To contribute to the project contact Juan Pablo del Peral 
at juandelperal@gmail.com
